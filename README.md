# CineVerse - Movie Review & Rating Website 🎬

**CineVerse** is a cinematic, responsive community-driven web application frontend built using React.js. Designed as an interactive platform for film enthusiasts, this project bridges the gap between passive movie info aggregation and active community interaction.

## 🚀 Core Features
- **Dynamic Review Engine:** Utilizes React state management (`useState`) to calculate movie audience score averages dynamically upon new critique submissions.
- **Spoiler Protection Layer:** Features dynamic interactive component masking that protects prospective plot points until explicitly clicked by the user.
- **Stripe Monetization Mockup:** Features an integrated functional micro-payment tipping modal workflow.
- **Advanced Admin Controls:** Includes dedicated runtime moderation buttons enabling quick content screening actions.
